# BetterControl+

**Minimalist monthly spending control. No spreadsheets. No complexity. Just clarity.**

BetterControl+ is a lightweight, browser-based financial tracker designed for individuals who want to monitor monthly expenses without the friction of formulas, accounts, or cloud services. Every entry stays on your device—powered entirely by local storage.

---

## ✦ Core Principles

- **Local-first privacy.** All data resides in your browser's `localStorage`. No servers, no tracking, no accounts.
- **Effortless entry.** Add spends with optional date, name, and required amount. Categories keep things organized.
- **Sophisticated simplicity.** A clean, typography-driven interface built with Tailwind CSS and vanilla JavaScript.

---

## ✦ Key Functionalities

### 📋 Monthly Overview
- View total spend for any selected month.
- Month selector automatically populates based on existing entries.
- Real‑time total updates as you add or remove items.

### ➕ Entry Management
- **Date (optional)** – defaults to today; supports any past or future date.
- **Name** – short description (e.g., “Netflix”, “Groceries”).
- **Category** – choose from built‑in categories or create your own on the fly.
- **Amount** – required positive numeric value.
- Delete any entry with a single click.

### 🗂️ Built‑in & Custom Categories
Default categories include:
- Online Subscriptions
- Health
- Rent
- Car‑related
- Food
- Utilities
- Entertainment
- Other

Select *“➕ new category”* from the dropdown to instantly add your own—permanently saved for future use.

### 💾 Import / Export
- **Export as JSON** – full backup of entries and categories.
- **Export as CSV** – compatible with spreadsheet software.
- **Import** – restore from a previously exported JSON or CSV file.

### 🧭 Seamless Navigation
- Clean intro screen with a single **Start** button.
- One‑click return to intro from the app header.
- Responsive layout that works on desktop, tablet, and mobile.
